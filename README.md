# Handwritten Character Recognition

## Project Overview
This project uses Machine Learning to recognize handwritten digits from images.

## Technologies Used
- Python
- Scikit-learn
- NumPy
- Matplotlib

## Workflow
1. Load handwritten digit dataset
2. Visualize images
3. Train Random Forest model
4. Evaluate accuracy
5. Predict handwritten digits

## Result
The model achieved high accuracy in recognizing handwritten digits.
